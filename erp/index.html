Administration ERP Odoo

Déploiement, configuration et maintenance de solutions de gestion intégrées.

Introduction

Dans le cadre de mon Master 1 SI, j'étudie la mise en œuvre technique des ERP. L'objectif est de transformer un serveur brut en une plateforme de gestion capable de supporter les processus métiers d'une entreprise (Ventes, RH, Inventaire).

Installation Technique

Installation d'Odoo 16 sur Ubuntu 24.04 en utilisant les scripts d'automatisation ou l'installation via le dépôt officiel.

# Mise à jour et installation des dépendances
sudo apt update && sudo apt install git python3-pip -y

# Configuration de PostgreSQL (Base de données)
sudo apt install postgresql -y
sudo su - postgres -c "createuser -s odoo16"

Gestion PostgreSQL

Un aspect crucial du système est la robustesse de la base de données. Je configure PostgreSQL pour assurer la persistance et l'intégrité des données d'entreprise.

  • Création d'utilisateurs dédiés.
  • Gestion des fichiers de configuration odoo.conf.
  • Planification des sauvegardes (Backups).
