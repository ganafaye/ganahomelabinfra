# 🌐 GanaHomeLab-Infra

Bienvenue sur mon infrastructure personnelle de déploiement. Ce projet sert de hub centralisé pour regrouper l'ensemble de mes travaux académiques (Master 1 Système d'Information) et mes projets personnels en **Cloud**, **DevOps**, **Virtualisation** et **ERP**.

---

## 🚀 Vision du Projet
L'objectif de cette plateforme est de démontrer mes compétences de **Développeur Python & Data Scientist** appliquées à la gestion d'infrastructures modernes. Elle est hébergée sur un environnement Linux et automatisée via un pipeline CI/CD.

## 🛠️ Stack Technique
* **Infrastructure :** VistaPanel (ByetHost), Linux/Ubuntu.
* **Automatisation :** GitHub Actions (Déploiement continu via FTP).
* **Conteneurisation :** Docker & OCI (Travaux pratiques AWS EC2).
* **ERP :** Administration et personnalisation d'Odoo avec PostgreSQL.
* **Analyse :** Intégration de modèles de Data Science (Streamlit, Scikit-Learn).

---

## 📂 Organisation du Hub
La plateforme est divisée en 4 piliers stratégiques :

1.  **[Cloud Computing](./cloud/)** : Travaux sur AWS, gestion des instances EC2 et architectures cloud.
2.  **[DevOps & Virtualisation](./devops/)** : Pipelines CI/CD, conteneurisation Docker et standards OCI.
3.  **[Systèmes Répartis](./systeme_repartis/)** : Cours et exercices sur la robustesse et la distribution des systèmes.
4.  **[ERP & Business Suite](./erp/)** : Gestion de flux de données sous Odoo et intégration de bases de données.

---

## 🔄 Pipeline CI/CD
Ce dépôt utilise **GitHub Actions** pour assurer un déploiement automatique :
* Chaque `git push` sur la branche `main` déclenche une synchronisation sécurisée vers le serveur de production.
* Gestion des variables d'environnement via les **GitHub Secrets** pour une sécurité optimale.
