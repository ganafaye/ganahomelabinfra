Docker & Conteneurisation

Standardisation des environnements de développement et de production.

1. Qu'est-ce que Docker ?

Docker est une plateforme permettant de lancer des applications dans des environnements isolés appelés conteneurs. Contrairement aux machines virtuelles, les conteneurs partagent le noyau du système hôte, ce qui les rend extrêmement légers et rapides.

2. Méthodes d'Installation

Ubuntu (Docker Engine)

Idéal pour les serveurs et les instances AWS EC2.

# Mise à jour des dépôts
sudo apt update
sudo apt install ca-certificates curl gnupg -y

# Ajout de la clé officielle Docker
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

# Installation du moteur
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin -y
Windows Desktop

Installation via l'exécutable **Docker Desktop**. Nécessite l'activation de **WSL 2** ou Hyper-V.

Télécharger l'installeur
macOS Desktop

Disponible pour puces Intel et **Apple Silicon (M1/M2/M3)** via Docker Desktop.

# Installation via Homebrew
brew install --cask docker

3. Vérification de l'Installation

Une fois installé, vérifiez que le daemon Docker répond correctement :

# Vérifier la version
docker --version

# Lancer le conteneur de test universel
sudo docker run hello-world

Sur Linux, n'oubliez pas d'ajouter votre utilisateur au groupe docker pour éviter d'utiliser sudo :
sudo usermod -aG docker $USER

Docker Desktop (Windows & Mac)

Docker Desktop est une application conviviale qui inclut Docker Engine, Docker CLI, Docker Compose et même un cluster Kubernetes local.

Pour Windows
  • Nécessite WSL 2 (Windows Subsystem for Linux).
  • Activer la virtualisation dans le BIOS.
  • Permet d'utiliser Docker directement dans le terminal PowerShell ou CMD.
Pour macOS
  • Compatible avec puces Intel et Apple Silicon (M1, M2, M3).
  • Utilise une machine virtuelle Linux légère et transparente.
  • Installation simple via fichier .dmg ou Brew.
